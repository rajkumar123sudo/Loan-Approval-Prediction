# Loan Approval Prediction

Machine Learning project to predict whether a loan application will be approved.

## Algorithms
- Logistic Regression
- Decision Tree
- Random Forest

## Accuracy

- Logistic Regression : 86.18%
- Random Forest : 82%
- Decision Tree : 74%

## Best Model

Logistic Regression

## Dataset

614 Records
13 Features
